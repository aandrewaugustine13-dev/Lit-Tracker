## Tips for Best Results

1. **Format Your Script Properly**
   ```
   INT./EXT. LOCATION - TIME
   
   Panel N Description.
   
   CHARACTER NAME
     Dialogue here.
   ```

2. **Use All-Caps for Character Names**
   - âœ… `ELI` will be detected
   - âŒ `Eli` might be missed

3. **Clear Slug-Lines for Locations**
   - âœ… `INT. APARTMENT - NIGHT`
   - âœ… `EXT. WASTELAND - DAY`
   - âŒ `At the apartment` (won't be detected)

4. **Enable LLM for Better Results**
   - Finds implicit mentions
   - Detects relationships
   - Higher total entity count
   - Costs ~0 (free Gemini tier)

5. **Review Confidence Scores**
   - Deselect items below 60% if unsure
   - Cross-reference context snippets
   - Check line numbers in your script
